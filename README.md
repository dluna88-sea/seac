# Sistema Estatal Anticorrupción

Sistema para gestionar apartado de transparencia del sitio web.

## Install Commands:


```bash
  git clone https://github.com/dluna88-sea/seac.git
  cd seac
  npm install
```

Ejecutar en modo desarrollo:
```bash
  npm run dev
```

Desplegar a producción:
```bash
  npm run build
```
